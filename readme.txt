With the Plus Lightbox you can add Lightbox attribute very simple to linked images or flash files in pages, posts and comments, it supporting grouping by ID.

== Description ==

The Plus Lightbox allows you to add custom attribute (a rel or class) to linked images in 
posts, pages, and comments, image formats supported are bmp, gif, jpeg, jpg and png. It can then be used with any of the popular 
lightbox plugins.

Is not including Javascript files for any lightbox plugins, these should be included yourself via your theme.

Group images for each post by adding [id] into your rel or class and add a custom attribute to 
linked Flash files (swf and flv).

== Installation ==

1. Unzip the archive and upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'The Plus Lightbox' in the settings menu
4. Add your custom attributes for images and flash files (both optional) and click save
5. For grouping use [id] for example `rel="lightbox[id]"`
